# Product Context

## Problema que resuelve

Permitir a usuarios (o evaluadores del lab) ver y seguir información de vuelos de forma clara.

## Experiencia de usuario esperada

- (Definir: búsqueda por número de vuelo, aeropuerto, mapa, lista en tiempo real, etc.)
- Interfaz simple y coherente con el stack elegido.

## Flujos principales (borrador)

1. Usuario abre la aplicación.
2. Usuario busca o selecciona un vuelo.
3. La app muestra detalle / estado del vuelo.
